## RUN TEST CASES

1. Setup the virtual environment by cd-ing to this directory

2. If no virtual environment is prepared, create one using `python -m venv <directory>`

3. Activate the environment with command:
    - MacOS/Linux: `source venv/bin/activate`
    - Windows: `.\.venv\Scripts\activate`

NOTE: ENSURE YOU HAVE THE FOLLOWING DEPENDENCIES: unittest, pydantic. You can install these using `pip install <dependency-name>`

4. Run the command `python -m unittest string_limits_tests.py` in the virtual environment to run the test cases.
