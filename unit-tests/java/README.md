# Java Unit Tests

- How to run these Java tests  
- install the Java Development Kit (JDK)  
- make sure you can run java and javac from the terminal  
- navigate to the unit-tests/java folder through the command "cd unit-tests/java"
- run the command "javac StringValidator.java TestStringValidator.java" to compile the source and test files  
- run the command "java TestStringValidator" to execute the unit test
